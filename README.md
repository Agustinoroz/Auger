# Auger

En este repositorio se encuentran los codigos de mi tesis de licenciatura en el Observatorio Pierre Auger.

Mi trabajo se divide en 3 ramas principales:
-El estudio de dos detectores que operaron como tanques de referencia 
-El analisis a largo plazo de uno de estos tanques en un periodo de operacion mayor
-El estudio de todos los detectores del arreglo.

Uno de los codigos más importantes ha sido alforitmo_de_estabilidad: un codigo que busca automaticamente intervalos de tiempo en los datos de mas de 1600 detectores y exitosamente identifica si la evolución de un parametro relacionado con sus constantes de claibracion evolucionó de manera estable en el tiempo en base a criterios definidos por mi.


A lo largo de este trabajo trabaje principalmente con python (armando codigos de analisis de datos) y C++ para leer y modificar los codigos de adqquisicion y calibracion desarrollados por la colaboracion.


Para el analisis de datos en python explore diferentes maneras de anailisis. He trabajado con librerias de estadistica, PELT, librerias para hacer ajustes, para calcular nubes de probabilidad, etc.
